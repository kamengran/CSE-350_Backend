CSE-350 Backend — Budget & Financial Aid Calculator

This is the backend for our **Budget Buddy / Financial Aid Project** built using **Python Flask**.  
It provides REST APIs for simple financial calculations (no user data is stored).

#######################
 Features
✅ Financial Aid Estimator  
✅ Budget Buckets (recommended monthly spending)  
✅ Savings Goal Planner  
✅ Compare Spending vs Budget (optional extension)
#######################

Tech Stack
- **Python 3.13**
- **Flask 3.x**
- **Flask-CORS**
- **PowerShell / cURL** for testing
- **Frontend:** HTML, CSS, JS (separate team)
